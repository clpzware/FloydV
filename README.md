## ğŸš€ Quick Start

### Prerequisites
- Java 17 or higher
- Minecraft installation
- Basic understanding of Minecraft modding

### Installation
1. Download the latest release from the [Releases](https://github.com/clpzware/FloydV/releases) page
2. Place the JAR file in your Minecraft mods folder
3. Launch Minecraft with Fabric installed
4. Join your preferred server and press **RIGHT SHIFT** to open the GUI

### Building from Source
```bash
# Clone the repository
git clone https://github.com/clpzware/FloydV.git
cd FloydV

# Build the project
./gradlew build

# The built JAR will be in build/libs/
```
